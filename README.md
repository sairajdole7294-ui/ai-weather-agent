
# Weather AI Agent 🌦️

A Python-based AI Weather Agent that fetches real-time weather data using the OpenWeather API.

## Features
- Get real-time weather information
- Shows temperature
- Shows weather condition
- Shows humidity
- Shows wind speed
- Uses REST API
- Built with Python

## Technologies Used
- Python
- Requests Library
- OpenWeather API

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/weather-ai-agent.git
```

2. Install the required library:
```bash
pip install requests
```

3. Add your OpenWeather API key in `weather.py`.

4. Run the program:
```bash
python weather.py
```
